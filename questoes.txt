3-O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R:o comando git add adiciona as mudancas para o proximo commit que sera realizado. 
Enquanto o git add <arquivo> é usado em torno de arquivos novos e onde nao tem onde um repositório podendo ser todo tipo de alteracao como novos, modificados e os deletados. 
Ja o git add e usado para adicionar arquivos novos e mmodificados mas nao os deletados.
